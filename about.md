# The Bull Up에 대하여
<br />
<br />

## 사이트 소개
**The Bull Up**은 퀀트 투자 전략을 활용하여 한국 주식 시장의 종목을 분석하고 매일 트레이딩 시그널을 제공하는 웹사이트입니다.<br />
인공지능(AI) 기반의 객관적이고 데이터 중심적인 분석을 통해 투자자들에게 참고용 정보를 제공합니다.
<br />

## 투자 전략

### 추적 손절을 적용한 이동평균선 교차 전략
본 사이트는 **추적 손절(Trailing Stop Loss)**을 적용한 **이동평균선 교차 전략**을 기반으로 합니다.<br />
이 전략은 다음과 같은 원리를 사용합니다:

#### 골드크로스 (Golden Cross)
- **의미**: 단기 이동평균선이 장기 이동평균선을 상향 돌파하는 신호
- **신호**: 상승 추세의 시작을 나타냄
- **행동**: 현금 → 주식 매수

#### 데드크로스 (Death Cross)
- **의미**: 단기 이동평균선이 장기 이동평균선을 하향 돌파하는 신호
- **신호**: 하락 추세의 시작을 나타냄
- **행동**: 주식 → 현금 매도

#### 추적 손절 (Trailing Stop Loss)
- **의미**: 주가가 상승할 때 손절가를 함께 상향 조정하는 손실 제한 전략
- **효과**: 수익을 보호하면서 손실을 최소화
- **행동**: 손절가 도달 시 주식 → 현금 매도

<br />
## 제공 정보

### 일일 트레이딩 시그널
- 매수 신호 (골드크로스 발생 종목)
- 매도 신호 (데드크로스 또는 손절 발생 종목)
- 각 종목의 신호 가격 정보
- 신호 유형별 분류

### 종목별 상세 분석
- 기술적 분석 결과
- 시그널 발생 시점 및 가격
- 포지션 변화 이력

<br />
## 개별 포스트 내용
각 종목 포스트는 다음과 같은 상세 정보를 포함합니다:

#### **포지션 변화 신호**
- 현재 포지션 상태 (주식 보유 또는 현금)
- 신호 유형 (골드크로스, 데드크로스, 손절)
- 신호 발생 시 종가 정보

#### **백테스팅 분석 결과**
- 최적 파라미터 탐색: 다양한 이동평균선 조합과 손절률을 테스트하여 최적의 파라미터를 찾습니다
- 수익률 비교: 전략 수익률과 Buy & Hold 수익률을 비교
- 성과 지표: 샤프 지수(Sharpe Ratio), 최대 낙폭(MDD), CAGR 등
- 거래 통계: 승률, 평균 손익, 손익비 등

#### **최적 파라미터 결과**
- 최고 수익률을 달성한 파라미터 조합
- 단기/장기 이동평균선 기간
- 추적 손절률 설정
- 상위 10개 파라미터 조합 비교

#### **거래 내역 상세**
- 모든 매수/매도 거래의 날짜, 가격, 수익률
- 각 거래의 보유 기간
- 매도 사유 (골드크로스, 데드크로스, 손절)
- 거래별 수익률 및 손익

#### **포지션 변화 이력**
- 시간순으로 정리된 모든 포지션 변화 기록
- 매수/매도 시점의 종가 정보
- 각 변화의 신호 유형

#### **데이터 분석 기간**
- 분석에 사용된 주가 데이터의 기간 (일반적으로 5년 이상)
- 데이터 다운로드 및 처리 정보

<br />
## 데이터 처리 방식

1. **실시간 데이터 수집**: 한국 주식 시장의 최신 데이터(전일 종가)를 수집
2. **AI 기반 분석**: 인공지능 알고리즘을 통한 객관적 분석
3. **백테스팅 검증**: 과거 데이터를 기반으로 한 전략 검증
4. **자동화된 시그널 생성**: 규칙 기반 시스템을 통한 일관된 시그널 생성
